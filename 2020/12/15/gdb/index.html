日常使用的gdb命令。

C++ gdb调试

命令参数

命令 简写 解释
file - 装入想要调试的可执行文件
list l 列出产生执行文件源代码的一部分
next n 执行一行源代码但不进入函数内部
step s 执行一行源代码而且进入函数内部
run r 执行当前被调试的程序
quit q 终止GDB
print p 打印变量
break b 在代码里设置断点
info break i b 查看断点信息（位置）
delete d 删除设置的断点
watch wa 监视一个变量的值，一旦值有变化，程序停住
help h 帮助指令
